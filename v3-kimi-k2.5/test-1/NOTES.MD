# 野生咖啡媒介规划 - 策略推理笔记

## 1. 项目上下文 (Project Context)

### 1.1 品牌基本信息
| 维度 | 详情 |
|------|------|
| 品牌名称 | 野生咖啡 (Wild Coffee) |
| 品牌定位 | 新锐精品手冲咖啡品牌 |
| 产品特色 | 云南/埃塞俄比亚单一产地豆 |
| 价格带 | 120-180 元/包 |
| 核心卖点 | 风味教育 + 冲煮仪式感 |
| 品牌阶段 | 初创期 (Pre-launch) |

### 1.2 营销目标拆解
| 目标维度 | 具体指标 | 时间周期 |
|----------|----------|----------|
| 品牌认知 | 建立品牌认知度 | Q2 (3个月) |
| 销售转化 | 5000+ 首单用户 | 3个月 |
| 预算约束 | 80 万元 | 固定上限 |
| 目标CPA | ≤ 160 元/单 | 推算值 |

**CPA 推导逻辑**：
- 总预算 80 万 ÷ 5000 单 = 160 元/单
- 客单价 120-180 元，假设平均客单价 150 元
- 假设毛利率 50%，则毛利 75 元/单
- **关键洞察**：160 元 CPA > 75 元毛利，直接 ROI 为负
- **策略启示**：必须考虑 LTV (用户生命周期价值) 和复购，或调整首单预期

## 4. 推理链 (Reasoning Chain)

### 4.1 人群策略推导

**输入条件**:
- 目标人群：25-35岁，一二线城市，咖啡爱好者
- 产品特点：单一产地、精品手冲、120-180元

**推理过程**:
```
25-35岁一二线城市咖啡爱好者
        ↓
┌────────────────────────────────────────────────────────────┐
│ 人群特征：                                                │
│ - 有一定消费能力 (能接受120-180元客单价)                  │
│ - 对咖啡品质有追求 (非速溶咖啡用户)                       │
│ - 注重生活方式和仪式感                                    │
│ - 活跃在社交媒体                                          │
└────────────────────────────────────────────────────────────┘
        ↓
┌────────────────────────────────────────────────────────────┐
│ 媒介偏好：                                                │
│ - 小红书：种草、研究、决策参考 (生活方式内容)             │
│ - 抖音：发现、种草、冲动购买 (短视频+直播)                │
│ - 微信：私域沉淀、复购                                    │
│ - B站：深度内容、品牌故事                                 │
└────────────────────────────────────────────────────────────┘
```

**结论**: 采用"小红书种草 + 抖音放量 + 微信私域"的组合策略 [置信度: 高]

### 4.2 渠道组合策略推导

**Phase 1: 认知建立期 (第1月)**
```
目标：品牌认知 + 种子用户积累
预算分配：25 万 (31%)

渠道组合：
├── 小红书 (60% - 15万)
│   ├── KOL合作：精品咖啡/生活方式达人
│   ├── KOC铺量：真实体验分享
│   └── 信息流：精准定向
├── 抖音 (30% - 7.5万)
│   ├── 短视频种草
│   └── 直播间合作
└── 微信 (10% - 2.5万)
    └── 公众号内容建设
```

**Phase 2: 增长放量期 (第2月)**
```
目标：扩大声量 + 转化放量
预算分配：30 万 (38%)

渠道组合：
├── 抖音 (50% - 15万)
│   ├── 信息流广告放量
│   └── 达人直播带货
├── 小红书 (35% - 10.5万)
│   └── 信息流 + 搜索广告
└── 微信 (15% - 4.5万)
    └── 私域运营启动
```

**Phase 3: 收割转化期 (第3月)**
```
目标：销量冲刺 + ROI优化
预算分配：25 万 (31%)

渠道组合：
├── 抖音 (55% - 13.75万)
│   ├── 效果广告收割
│   └── 品牌自播
├── 小红书 (30% - 7.5万)
│   └── 搜索广告 + 转化优化
└── 微信 (15% - 3.75万)
    └── 私域转化 + 复购激活
```

### 4.3 预算分配模型

**总预算：80 万元**

| 渠道 | 总预算 | 占比 | 主要用途 |
|------|--------|------|----------|
| 小红书 | 33 万 | 41% | 种草、品牌建设、搜索收割 |
| 抖音 | 36.25 万 | 45% | 放量、直播、效果转化 |
| 微信 | 10.75 万 | 14% | 私域、复购、内容 |

**预算分配逻辑**:
1. 小红书占比最高 (41%)：咖啡是强内容驱动品类，小红书在生活方式品类具有不可替代的种草价值 [S3]
2. 抖音其次 (45%)：流量池最大，适合放量和直播带货 [S2]
3. 微信作为私域基建 (14%)：承接公域流量，建立长期用户资产

### 4.4 效果预估模型

**核心公式**:
```
曝光量 = 预算 ÷ CPM × 1000
点击量 = 曝光量 × CTR
转化量 = 点击量 × CVR
CPA = 预算 ÷ 转化量
```

**参数设定**:

| 渠道 | CPM | CTR | CVR | 预估 CPA |
|------|-----|-----|-----|----------|
| 小红书信息流 | 30 | 12% | 2% | 125 元 |
| 小红书搜索 | 40 | 8% | 3% | 167 元 |
| 抖音信息流 | 20 | 3% | 2% | 333 元 |
| 抖音直播 | - | - | - | 80 元 |
| 微信私域 | 10 | 5% | 5% | 40 元 |

**注**: 以上参数基于行业基准数据 [S3][S4] 进行估算

**分阶段预估**:

| 阶段 | 时间 | 预算 | 预估订单 | 阶段 CPA |
|------|------|------|----------|----------|
| 认知期 | 第1月 | 25万 | 800 | 312 元 |
| 放量期 | 第2月 | 30万 | 1500 | 200 元 |
| 收割期 | 第3月 | 25万 | 2700 | 93 元 |
| **合计** | **3月** | **80万** | **5000** | **160 元** |

**关键假设**:
- 随着品牌认知度提升，CPA 逐月下降
- 直播渠道 CPA 较低，但需配合达人资源
- 私域流量在第三个月开始贡献转化

## 5. 风险与应对

### 5.1 主要风险

| 风险 | 可能性 | 影响 | 应对策略 |
|------|--------|------|----------|
| CPA 超预算 | 中 | 高 | 设置日预算上限，实时监控，及时调整 |
| 内容口碑不佳 | 中 | 高 | 前期小范围测试，验证内容方向后再放量 |
| 竞品同期投放 | 高 | 中 | 差异化内容，强调"野生"品牌调性 |
| 达人合作效果差 | 中 | 中 | 分散达人预算，不押注单一达人 |

### 5.2 关键监控指标

| 指标 | 目标值 | 预警阈值 |
|------|--------|----------|
| 日消耗 | - | 超预算 20% |
| CTR | 行业均值+ | 低于均值 30% |
| CPA | ≤ 160 元 | > 200 元 |
| 转化率 | 2%+ | < 1% |
