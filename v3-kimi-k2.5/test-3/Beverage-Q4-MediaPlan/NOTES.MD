## 3. 核心问题分析：为什么去年的 300 万"看起来效果不错，但销量不涨"？

### 3.1 "种草 1.2 亿人次"的数字拆解

**Agency 报告逻辑**:
```
总曝光量 = 1.2 亿人次
```

**审计拆解**:
```
1.2 亿人次 = 笔记阅读量 + 视频播放量 + 话题曝光量
         = 可能包含大量重复计算
         = 可能包含 3 秒以下无效阅读
         = 可能包含机器人/刷量账号
```

**水分来源分析**:

| 水分类型 | 说明 | 预估去水率 |
|---------|------|-----------|
| 重复计算 | 同一用户多次阅读被多次计数 | 30-50% |
| 无效曝光 | 3 秒以下快速划过 | 20-40% |
| 机器人/刷量 | 虚假账号产生的数据 | 5-15% |
| 自然流量混入 | 将平台自然推荐算作投放效果 | 难以估算 |

**真实触达估算**:
```
1.2 亿人次 × (1-40% 重复) × (1-30% 无效) × (1-10% 刷量)
≈ 1.2 亿 × 0.6 × 0.7 × 0.9
≈ 4536 万【真实有效曝光】
```

**关键洞察**: Agency 报告的 1.2 亿人次，真实有效曝光可能只有 **35-40%**。

### 3.2 销量增长 8% 的真相

**问题**: 4536 万真实曝光 → 销量增长 8%，这个转化合理吗？

**计算**:
```
假设:
- 产品客单价: 8 元
- 毛利率: 50%（即 4 元毛利/瓶）
- 曝光→点击转化率: 15% (行业平均 CTR)
- 点击→购买转化率: 2% (行业保守估计)

理论销量 = 4536 万曝光 × 15% CTR × 2% CVR
         = 13.6 万单

理论 GMV = 13.6 万 × 8 元 = 108.8 万元

理论毛利 = 13.6 万 × 4 元 = 54.4 万元

ROI = 54.4 万 / 300 万 = 0.18 (严重亏损!)
```

**结论**: 即使按照"包装数字"的 40% 真实曝光计算，300 万投入带来 54.4 万毛利，**ROI 为 0.18，严重亏损**。

这解释了为什么 Agency 只敢说"种草人次"，不敢说"实际销量"——因为转化链路断裂，曝光没有有效转化为购买。

## 4. 推理链：2000 万 Q4 媒介规划

### 4.1 营销目标拆解

```
总预算: 2000 万
目标: 品效协同

分配原则:
├─ 效果广告 (直接转化): 60% = 1200 万
├─ 品牌广告 (心智建设): 30% = 600 万
└─ 内容种草 (口碑沉淀): 10% = 200 万
```

### 4.2 渠道组合策略

基于 S1、S2、S3 的证据，Q4 饮料品类最优渠道组合:

| 渠道 | 预算分配 | 角色定位 | 核心 KPI |
|------|---------|---------|---------|
| 抖音 (效果) | 600 万 | 主转化阵地 | ROAS ≥ 2.5 |
| 抖音 (品牌) | 300 万 | 大促爆破 | CPM、互动率 |
| 天猫/京东 | 400 万 | 电商收口 | ROI、CTR |
| 小红书 | 200 万 | 种草蓄水 | 真实 CPS (单次搜索成本) |
| 微信生态 | 200 万 | 私域沉淀 | 加粉成本、复购率 |
| B 站 | 100 万 | 年轻人群渗透 | 完播率、互动率 |
| 分众/户外 | 200 万 | 线下心智 | 覆盖频次 |

### 4.3 抖音 800 万 → 5000 万 GMV 的审计

**Agency 承诺**: 投 800 万，带来 5000 万 GMV，ROAS = 6.25

**审计分析**:
```
ROAS = 6.25 意味着:
- 每投入 1 元广告费，带来 6.25 元 GMV

饮料行业合理 ROAS 区间: 2.5-4 (S5)

Agency 的 6.25 高出行业均值 56-150%

可能的情况:
1. 包含了大量自然流量 GMV（非广告贡献）
2. 使用了"归因窗口期"作弊（30 天内点击过广告都算）
3. 刷单/虚假订单
4. 极低客单价冲量（如 1 元试饮装）
```

**可审计版本**:
```
保守预估 (基于行业均值 ROAS 3):
- 800 万投入 → 2400 万 GMV

乐观预估 (基于品类红利期 ROAS 4):
- 800 万投入 → 3200 万 GMV

Agency 承诺的 5000 万 GMV 属于"极限 optimistic"情景，
需要额外条件支撑:
- 产品客单价 ≥ 15 元
- 毛利率 ≥ 60%
- 直播间转化率 ≥ 5%
- 无任何退货
```

### 4.4 小红书 200 万预算的真实预期

基于 S2 的 CTR 数据，重新计算:

```
预算: 200 万
小红书 CPM 预估: 80-150 元 (食品饮料类目)

曝光量 = 200 万 ÷ 100 元 × 1000 = 2000 万曝光

考虑去水率 60% (参考第 3 节分析):
真实曝光 ≈ 1200 万

按行业 CTR 15.67%:
点击量 ≈ 188 万

按 3% 搜索转化率:
搜索量 ≈ 5.6 万

按 10% 购买转化率:
销量 ≈ 5600 单

假设客单价 8 元:
GMV ≈ 4.5 万元

结论: 小红书的核心价值不是直接 ROI，而是「搜索心智」和「长效种草」。
      用 ROI 衡量小红书是不合适的，应该用「搜索成本」和「内容资产沉淀」衡量。
```

## 5. 结论卡片

### 5.1 预算分配结论 (Confidence: ⭐⭐⭐⭐)

| 渠道 | 预算 | 占比 | 预期 ROAS | 审计要点 |
|------|------|------|-----------|---------|
| 抖音效果 | 600 万 | 30% | 2.5-3.5 | 区分广告 GMV vs 自然 GMV |
| 抖音品牌 | 300 万 | 15% | - | 关注 CPM、互动成本 |
| 天猫/京东 | 400 万 | 20% | 3-5 | 关注 CTR、CVR、退货率 |
| 小红书 | 200 万 | 10% | 0.3-0.5 | 关注 CPS、内容沉淀量 |
| 微信 | 200 万 | 10% | 2-4 | 关注加粉成本、LTV |
| B 站 | 100 万 | 5% | - | 关注完播率、品牌好感度 |
| 线下 | 200 万 | 10% | - | 关注覆盖频次 |

### 5.2 数字真实性审计框架 (Confidence: ⭐⭐⭐⭐⭐)

**"包装数字"识别清单**:

| 指标类型 | 常见包装手法 | 审计方法 |
|---------|-------------|---------|
| 曝光量 | 计入 3 秒以下无效曝光 | 要求提供"有效曝光"数据 |
| 触达人次 | 重复计算同一用户 | 要求提供 UV 数据 |
| GMV | 包含自然流量、退货未剔除 | 要求提供广告归因 GMV + 退货率 |
| ROAS | 使用 30 天归因窗口 | 要求提供 1 天/7 天/30 天 ROAS 对比 |
| 种草人次 | 混入自然流量曝光 | 要求区分付费 vs 自然流量 |

### 5.3 核心建议

1. **所有承诺必须带边界条件**: Agency 说"800 万带来 5000 万 GMV"，必须追问"在什么客单价、什么毛利率、什么退货率、什么归因窗口下"

2. **小红书不要只看 ROI**: 小红书的核心价值是"搜索心智"，应该用 CPS (Cost Per Search) 衡量，而非 ROAS

3. **建立三方数据对账**: 要求 Agency 数据与电商平台后台数据、第三方监测数据三方对账

4. **设置阶梯付款**: 根据 KPI 达成情况设置阶梯付款，而非一次性预付
