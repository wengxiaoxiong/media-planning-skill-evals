# 项目上下文

- 项目：智能合同管理系统（B2B SaaS）6 个月媒介规划，目标客户为中大型企业法务部门。[S1]
- 已知挑战：
  - 品类教育不足，约 70% 决策者对“智能合同管理系统”认知不足。[S1]
  - 市场垂直且公开样本少，标准化行业基准有限。[S1]
  - 预算仅 30 万，且销售周期 3-6 个月，要求“教育 + 获客”并行。[S1]
- 业务目标：6 个月获得 50 个 MQL，并建立“行业专家”认知。[S1]

# 分析问题与范围界定

- 时间范围：2026-02-12 至 2026-08-11（6 个月）。
- 渠道范围：微信生态、搜索、垂直媒体、抖音/B站、小红书、再营销闭环。
- 目标边界：
  - 本次以 MQL 作为核心北极星指标，销售结果以“管道价值（Pipeline）”做二级追踪。
  - 因缺少品牌历史实测（各渠道 CPM/CTR/CVR/线索到商机转化），所有前瞻数字标记为“估算”或“参考”。
  - 报告中的平台趋势采用近 12 个月公开资料；超过 12 个月信息不作为关键锚点。

# 推理链（结论 -> 证据 -> 推导）

## 1) 营销目标拆解（Objective Decomposition）

- 结论 C1：本项目应采用“双目标串联”而非二选一：
  - 上层目标：品类教育（认知与信任建立）
  - 下层目标：MQL 获取（可落入销售漏斗）
  证据：[S1][S4]
  推导：目标同时要求“行业专家认知 + 50 MQL”，且市场仍需教育，必须先解决认知再放大收口。

## 2) 人群策略（Audience Strategy）

- 结论 C2：核心人群拆为三层，并采用不同内容资产承接：
  - 决策层（法务负责人/总法律顾问）：关注合规、风险、组织效率
  - 影响层（法务经理/采购/IT）：关注流程落地、系统集成、实施成本
  - 使用层（法务专员/合同管理员）：关注易用性、模板效率、审批体验
  证据：[S1][S3]
  推导：3-6 个月销售周期意味着多人共决；若只打单一角色，线索到商机转化会失真。

## 3) 渠道组合策略（Channel Mix）

- 结论 C3：建议“微信生态 + 搜索”做线索主引擎，“垂直媒体 + 视频内容”做教育引擎，“再营销”做转化提效。
  证据：[S4][S5][S6][S7]
  推导：头部平台流量仍集中，但 B2B 线索质量依赖深内容与多触点重复触达，需构建内容到留资的闭环。

## 4) 预算分配模型（Budget Allocation）

- 结论 C4：30 万预算建议如下（基准情景）：
  - 微信生态（公众号/视频号/企微）：35%（105,000）
  - 搜索广告（品牌词+问题词+竞品词）：25%（75,000）
  - 垂直媒体/行业社群/白皮书联投：15%（45,000）
  - 抖音+B站（专家内容分发）：12%（36,000）
  - 小红书（品牌信誉与雇主口碑补充）：5%（15,000）
  - 再营销与营销自动化：8%（24,000）
  证据：[S1][S2][S4][S5][S7]
  推导：预算受限下必须优先“高意图渠道”，同时保留教育与复访预算，避免只做末端收割导致线索枯竭。

## 5) 投放节奏与排期（Flight Planning）

- 结论 C5：6 个月采用“2-2-2”节奏（认知起盘 -> 线索放大 -> 收口优化）：
  - M1-M2：27%（内容资产与权威背书建立）
  - M3-M4：40%（主推线索获取与研讨会）
  - M5-M6：33%（再营销、销售协同、重点账户突破）
  证据：[S1][S4]
  推导：销售周期 3-6 个月决定前期必须完成教育与种子线索沉淀，中后段才有稳定 MQL 产出。

# 可复算模型（MQL 口径）

基础公式：

- 曝光量 = 预算 / CPM * 1000
- 点击量 = 曝光量 * CTR
- MQL = 点击量 * CVR(MQL)
- CPL = 预算 / MQL
- 新增 ARR = MQL * MQL->Won 转化率 * 平均年合同额
- ROI(ARR) = 新增 ARR / 预算

## 结果区间（估算）

- 保守情景：42 MQL（目标完成率 84%）
- 基准情景：61 MQL（目标完成率 122%）
- 乐观情景：79 MQL（目标完成率 158%）

## ROI(ARR) 区间（估算）

假设：平均年合同额 120,000 元（来自用户给定区间 8-20 万的中位近似）、MQL->Won 转化率分别为 8%/10%/13%。[S1]

- 保守：ARR = 42 * 8% * 120,000 = 403,200；ROI(ARR)=1.34
- 基准：ARR = 61 * 10% * 120,000 = 732,000；ROI(ARR)=2.44
- 乐观：ARR = 79 * 13% * 120,000 = 1,232,400；ROI(ARR)=4.11

> 注：因销售周期可达 6 个月，部分 ARR 可能在观察窗后确认；因此应同步跟踪 Pipeline 与 Win Rate。

# 结论卡片（含 confidence）

- 卡片 K1：在 30 万预算约束下，单纯品牌曝光或单纯效果收割都不足以达成目标，应执行“教育+线索”双轮驱动。`confidence: 高` [S1][S4]
- 卡片 K2：基准模型可支撑 50 MQL 目标（预计 60.9），但对搜索词策略和再营销运营执行依赖高。`confidence: 中高` [S1]
- 卡片 K3：微信生态与搜索应承担主预算（60%），短视频与内容平台承担认知与再触达，不宜反客为主。`confidence: 中高` [S4][S5][S7]
- 卡片 K4：竞品已在“AI 合同”叙事上推进，若本项目不建立专家内容壁垒，将在认知端持续被动。`confidence: 中` [S8][S9]

# 数据缺口与待验证事项

1. 历史真实漏斗缺失：渠道级 CPM/CTR/CVR、线索去重、MQL 评分口径尚未统一。
2. 销售协同数据缺失：MQL->SQL、SQL->Won 转化率缺乏历史分层（行业/客群/区域）。
3. 内容资产效率未知：白皮书、案例、研讨会等资产的下载率与留资率缺少基线。
4. 竞品买量强度缺失：目前仅有公开动态，缺少竞品分渠道预算与声量份额。
5. 待验证实验：
   - 关键词结构 AB：品牌词/问题词/竞品词预算配比
   - 落地页 AB：单页留资 vs 白皮书下载留资
   - 再营销窗口：7 天 / 14 天 / 30 天触达频次
