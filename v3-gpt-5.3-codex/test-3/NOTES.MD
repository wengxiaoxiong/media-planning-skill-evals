# 推理链（结论 -> 证据 -> 推导）

## 1) 营销目标拆解（Objective Decomposition）

- 结论 C1：Q4 目标应为“双目标体系”：
  - 品牌侧：有效触达、有效互动、品牌搜索提升
  - 生意侧：可归因 GMV、CPA、利润 ROI
  证据：[S1][S5][S6]
  推导：在消费总盘温和增长、饮料品类增速有限背景下，仅追求曝光会放大“报量不报效”风险。

## 2) 人群策略（Audience Strategy）

- 结论 C2：应采用“核心购买人群 + 扩展种草人群”分层策略，并做频控与去重。
  证据：[S1][S5]
  推导：历史案例显示高人次曝光未转化为等比例销量增量，说明人群匹配与链路承接存在断点。

## 3) 渠道组合策略（Channel Mix）

- 结论 C3：建议“短视频转化 + 内容种草 + 私域复购 + 搜索收口”的组合，而非单平台押注。
  证据：[S3][S4][S7]
  推导：头部平台具备规模流量与成熟广告系统，但单平台归因易高估；多渠道协同更利于审计与纠偏。

## 4) 预算分配模型（Budget Allocation）

- 结论 C4：2,000 万基准分配建议：
  - 抖音 40%（800 万）
  - 小红书 20%（400 万）
  - 微信生态 15%（300 万）
  - B站 10%（200 万）
  - 快手 7.5%（150 万）
  - 搜索/电商站内 5%（100 万）
  - 测试与归因 2.5%（50 万）
  证据：[S1][S2][S3][S4][S7]
  推导：保证转化主阵地占比，同时保留种草与复购闭环，并预留“实验预算”修正参数偏差。

## 5) 投放节奏与排期（Flight Planning）

- 结论 C5：Q4 节奏建议为 25% / 45% / 30%（10 月蓄水、11 月爆发、12 月复购回补）。
  证据：[S1][S6]
  推导：11 月大促节点集中，12 月须延续礼赠与复购场景，避免爆发后快速衰减。

## 6) 争议数据审计判定（True vs Packaged）

- 结论 C6：`抖音 800 万 -> 5,000 万 GMV` 为“高目标陈述”，不能直接判假，但不能无审计直接采信。
  证据：[S1][S6][S9]
  推导：
  - 该陈述隐含 ROI(GMV)=6.25x。
  - 按漏斗公式反推：
    - 若 CPM=40、CTR=1.8%、CVR=3.5%，则需 AOV 约 397 元；
    - 若 CVR 提升到 5%，仍需 AOV 约 278 元。
  - 对典型饮料品类而言，上述客单要求偏高，除非有高价礼盒/多件组合/跨品类打包。
  判定：**高概率存在“包装空间”**，需补交明细后才能确认真实性。

- 结论 C7：`小红书 300 万 -> 1.2 亿人次` 在曝光口径上“可能成立”，但不等于业务结果。
  证据：[S1][S2][S9]
  推导：
  - 反推 CPM=25（=300万/1.2亿*1000）。
  - 相比食品饮料行业均值 CPM=47.4（参考值），25 并非不可能，但通常需要大量低成本流量与达人结构配合。
  - “人次”非“去重人数”，也非“有效触达”或“成交贡献”。
  判定：**曝光陈述可作为参考，但商业价值陈述不充分**。

# 结论卡片（含 confidence）

- K1：媒介汇报体系必须从“曝光/阅读”升级为“漏斗+利润”双轨：`曝光->点击->转化->GMV->毛利`。`confidence: 高` [S1][S9]
- K2：Q4 采用多渠道协同比单渠道押注更稳健，且更便于归因交叉验证。`confidence: 中高` [S3][S4][S7]
- K3：两条争议陈述均存在“可发生但可包装”的特征，未经原始明细不得用于业绩结论。`confidence: 高` [S1][S2][S9]
- K4：在当前信息条件下，媒体直接归因 ROI(GMV)更可能落在 0.43x-1.52x，6.25x 需异常优参数与客单结构。`confidence: 中` [S1][S2][S6]

# 可复算模型（用于 CFO 审计）

- 曝光量 = 预算 / CPM * 1000
- 点击量 = 曝光量 * CTR
- 转化量 = 点击量 * CVR
- CPA = 预算 / 转化量
- 归因 GMV = 转化量 * 客单价
- ROI(GMV) = 归因 GMV / 预算
- ROI(Profit) = 归因 GMV * 毛利率 / 预算

# 数据缺口与待验证事项

1. 商品结构：各 SKU 客单价、毛利率、组合装占比（决定 ROI 上限）
2. 归因规则：点击归因窗口、曝光归因窗口、跨平台去重逻辑
3. 历史实测：去年 Q4 分渠道 CPM/CTR/CVR/复购率、订单明细
4. 真伪审计要求：
   - 明细级投放日志（time, creative, audience, cost）
   - 明细级订单日志（order_id, source, timestamp, sku, margin）
   - 去重与归因脚本版本（可复核）
5. 纠偏机制：
   - 周度偏差阈值：`|实际-预测|/预测 > 20%` 触发调仓
   - 预留 10%-15% 预算做 A/B（素材、人群、出价）
