# Media Plan Skill (媒介规划技能库)

这是一个用于开发、测试和存储“媒介规划”（Media Planning）报告生成技能的专业仓库。该技能旨在通过 AI 模型生成咨询级别的媒介策略报告，辅助品牌方和代理商进行市场决策。

## 核心理念

本项目基于“先推理，后展示”的双阶段流程，确保生成的报告不仅视觉精美，且逻辑严密、数据可溯源。

- **透明推理**: 所有结论必须在 `NOTES.MD` 中通过“证据链”推导得出。
- **叙事驱动**: 报告不再是模板化的堆砌，而是根据项目核心挑战构建商业故事。
- **咨询级视觉**: 生成响应式 HTML 报告，集成可交互的 ECharts 图表，风格对齐 McKinsey/BCG 咨询标准。

## 工作流

该技能严格执行以下工作流：

1.  **Phase A: Research + Reasoning (研究与推理)**
    *   **信息抽取**: 提取品牌、产品、目标、预算、受众等关键维度。
    *   **证据收集**: 检索行业趋势、竞品动态及平台数据，记录至 `Evidence Ledger`。
    *   **逻辑推导**: 基于证据链进行人群策略、渠道组合及预算分配的推演。
    *   **产出**: 生成项目文件夹下的 `NOTES.MD`。

2.  **Phase B: Narrative + Visualization (叙事与渲染)**
    *   **叙事构建**: 根据 `NOTES.MD` 中的核心挑战构建报告逻辑。
    *   **视觉渲染**: 遵循 `Style.MD` 规范生成单文件 HTML 报告。
    *   **交互集成**: 注入 ECharts 图表展现预算、节奏、效果预估等数据。
    *   **产出**: 生成项目文件夹下的 `report.html`。

## 技术规范

- **图表库**: ECharts 5.5.0 (通过 CDN 引入)
- **样式**: 原生 CSS (响应式布局，支持 PC 与手机端)
- **数据标准**: 引用编号对齐 (`[S1]`, `[S2]`, ...)，支持全流程溯源。
